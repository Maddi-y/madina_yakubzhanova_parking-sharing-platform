Parking Sharing Platform

Parking Sharing Platform — веб-приложение для поиска, бронирования и размещения парковочных мест.

Пользователи могут:

зарегистрироваться и войти в систему;
создавать объявления о парковочных местах;
просматривать доступные парковочные места;
бронировать парковочные места на выбранное время;
добавлять парковочные места в избранное;
оставлять отзывы после завершения бронирования.

Проект разработан в рамках итогового задания с использованием технологий Java, Spring MVC и JDBC.
